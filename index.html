🧬

Why Mitochondrial DNA?

mtDNA has a high mutation rate, a small genome (~16 kb), and maternal inheritance — making it ideal and widely standardized for phylogenetic studies. These properties enable clear differentiation between related species without recombination noise.

🎯

Why These 7 Species?

All Esocidae species with available complete mitochondrial genomes were selected. Dallia and Novumbra serve as outgroups to root the tree, representing the broader order Esociformes but sister to the Esox genus.

Workflow
Methodology

A rigorous bioinformatics pipeline was followed — from raw genome data through alignment, model selection, tree construction, bootstrapping and statistical validation.

1

Genome Data Collection (Muhammad)

Full mitochondrial genome sequences (FASTA format) were searched on NCBI Nucleotide for each Esocidae species. First complete-genome results were selected and concatenated into a single multi-FASTA file.

2

Multiple Sequence Alignment — MAFFT

All sequences uploaded to MAFFT online server with default settings. Only the uppercase/lowercase option was changed to "same as input". Output copied into aligned.fasta (123 KB aligned vs 121 KB unaligned).

3

Tree Construction — MEGA 12

Maximum-likelihood tree built using MEGA12's Phylogeny module. Sequence registered as nucleotide (non protein-coding) since full mitochondrial genome contains non-coding regions.

4

Model & Parameter Selection

Substitution model selected: Gamma-distributed rates + Invariant sites (G+I). This accounts for rapid mitochondrial mutation rates and high reproductive variation in fish populations. Bootstrapping enabled for statistical validation.

5

Visualization & Export

Two tree versions exported: one with bootstrap values and one with branch lengths. Final editing done in MSPaint. R code generated heatmap, GC-content plots, ANOVA boxplot, and distance CSV.

⚙️

Why Maximum Likelihood?

Compared to distance-based (UPGMA/NJ) and parsimony methods, Maximum Likelihood:

Statistically most robust
Handles unequal substitution rates
Works well with 6–7 taxa
Widely accepted in literature
G+I model for mitochondrial DNA

Research Background
Phylogenetic Methods Comparison
📏

Distance-Based (NJ/UPGMA) (Chosen)

Computes pairwise distances and clusters by similarity. Fast but loses character-level information. Used here as a comparison baseline via pairwise distance heatmap.

🔺

Maximum Parsimony

Selects the tree requiring fewest evolutionary changes. Computationally intensive for large datasets. Considered but not chosen — ML preferred for accuracy with whole mtDNA.

✅

Maximum Likelihood

Selects the statistically most probable tree given a DNA mutation model. Character-based, handles rate variation, widely used. Combined with Gamma+I substitution model.

Results & Analysis

Key findings from the ML tree construction, pairwise genetic distance matrix, and R-based statistical analyses.

🌳 Tree Topology

The ML tree places E. americanus and E. niger as closest relatives (distance ≈ 0.0043), grouped with E. lucius and E. reichertii. E. masquinongy is a more distant Esox. Novumbra and Dallia form clear outgroups, with Dallia being the most distant (~0.197–0.201).

📐 Bootstrap Validation

Bootstrapping was used to statistically validate phylogenetic groupings. High bootstrap values at key nodes indicate strong support for the recovered topology. Two tree images exported: one highlighting bootstrap values and one showing branch lengths for interpretation.

Statistical Tests — GC Content Analysis
📈

Nucleotide Composition

Stacked bar chart showing ATGC composition (%) per species. Mitochondrial genomes generally show AT-bias. Gaps from alignment were excluded before calculating percentages using seqinr in R.

🧮

One-Way ANOVA on GC%

Each genome was split into 1000 bp windows to generate multiple GC% values per species. ANOVA tests whether species significantly differ in genome GC content — relevant to genome stability and evolutionary pressure.

🔍

Tukey's HSD Post-Hoc

If ANOVA was significant (p < 0.05), Tukey's HSD was applied to identify which specific species pairs differ significantly, providing pairwise statistical comparison of GC content across all 21 pairs.

Implementation
R Visualization Code

The visualization and statistical analysis were implemented in R using ape, seqinr, ggplot2 and reshape2 packages. The script produces 4 plots, a CSV, and ANOVA/Tukey output files.

Discussion
Interpretation & Conclusions

Evolutionary insights drawn from the phylogenetic tree, distance matrix, and genomic analysis — interpreted in the context of known Esocidae biology.

🐟

Evolutionary Relationships

The ML tree confirms E. americanus and E. niger as sister taxa (distance 0.0086), suggesting they shared a common ancestor most recently. E. lucius and E. reichertii also cluster, while E. masquinongy (Muskellunge) is more diverged within Esox. Dallia is the deepest outgroup.

🌍

Habitat & Geographic Context

Northern Pike (E. lucius) and Amur Pike (E. reichertii) have a broad Holarctic distribution across North America, Europe and Asia — consistent with their phylogenetic proximity. E. americanus is eastern North America, while E. niger is widespread across eastern US, reducing overlap-based divergence pressure.

⏱️

Divergence Time Considerations

Fish have high reproductive rates and relatively short generation times, increasing genetic drift. Mitochondrial genomes accumulate mutations faster than nuclear DNA, making them sensitive markers for divergence even over moderate evolutionary timescales.

🔮

Comparison: ML vs Distance Tree

Pairwise distances show similar groupings to the ML tree but lack statistical model-based support. Distance-based trees (NJ) would use uniform substitution rates and miss rate variation captured by the Gamma+I model — potentially distorting deeper nodes.

Further Research Directions
🧬

Nuclear Genome Comparison

Compare mtDNA-based topology with nuclear genome phylogenies to resolve any discordance due to incomplete lineage sorting or mitochondrial introgression.

🌿

Ecological Trait Mapping

Map ecological traits (habitat type, depth preference, prey) onto the phylogeny to test for niche conservatism versus adaptive divergence across the Esocidae lineage.

➕

Outgroup Expansion

Add additional outgroup species (e.g. from Salmoniformes or Umbridae) and include Esox cisalpinus if genome becomes available to improve tree rooting and species coverage.

Presentation Insights
Key Points for Academic Presentation
Use the aligned FASTA file to visually demonstrate mutations and gaps (InDels, frameshifts) — these are biologically meaningful artifacts of genome evolution.
Explain why mtDNA: high mutation rate, 16kb genome, maternal inheritance, no recombination, widely standardized in fish phylogenetics literature.
Show workflow diagram (NCBI → FASTA → MAFFT → MEGA12 → R analysis) to make the pipeline clear for the audience.
Highlight the conserved mitochondrial regions visible in the alignment — these represent functionally constrained genes (e.g. cytochrome b, ribosomal RNA) shared across all Esocidae.
Explicitly compare the ML tree with the MAFFT distance tree to demonstrate how method choice affects topology, particularly at deeper nodes.
Future direction: mention potential for isotope analysis or fossil calibration to assign absolute divergence dates to the tree nodes.
